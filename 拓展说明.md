# sobel_02_hdmi_sobel 扩展说明

[hdmi_sobel_display.v](sobel_02_hdmi_sobel.srcs/sources_1/new/hdmi_sobel_display.v) 已通过 `DISPLAY_MODE` 参数支持 2 种扩展显示模式。
---

## 快速使用

打开 [hdmi_sobel_display.v](sobel_02_hdmi_sobel.srcs/sources_1/new/hdmi_sobel_display.v)，修改第 31 行 `DISPLAY_MODE` 的值：

| DISPLAY_MODE | 模式 | HDMI 效果 |
|-------------|------|-----------|
| `2'd0` | 原始灰阶（默认） | 黑底白边灰阶图 |
| `2'd1` | 边缘反色 | 白底黑边 |
| `2'd2` | 彩色边缘标记 | 红/绿/蓝伪彩色 |


---

## 扩展一：边缘反色显示

**设置**：`DISPLAY_MODE = 2'd1`

### 原理

对 Sobel 输出的 8-bit 边缘强度按位取反（`~` 运算符），`output = 255 - edge_pixel`：

```text
原显示（MODE 0）：edge_pixel 大 → 亮（白边黑底）
反色后（MODE 1）：edge_pixel 大 → 暗（黑边白底）
```

### 关键代码（第 129-130 行）

```verilog
wire [7:0] edge_invert;
assign edge_invert = ~edge_pixel;   // 按位取反，等价于 255 - edge_pixel
```

### RGB 映射关系

| edge_pixel | 输出值 | 显示效果 |
|------------|--------|----------|
| 0（无边缘） | 255 | 白色背景 |
| 64 | 191 | 浅灰 |
| 128 | 127 | 中灰 |
| 192 | 63 | 深灰 |
| 255（最强边） | 0 | 黑色边缘 |

### 预期现象

HDMI 显示**白底黑边**图像，与原始实验的黑底白边完全相反。边缘越强越黑。

### 验收标准

- [ ] HDMI 显示白底黑边图像
- [ ] 报告中写出映射关系：`output_R/G/B = 255 - edge_pixel`

## 扩展二：彩色边缘标记

**设置**：`DISPLAY_MODE = 2'd2`

### 原理

按 Sobel 边缘强度分为三级，输出到不同颜色通道：

```text
强边缘 (edge_pixel >= 192) → 红色通道 (R=edge_pixel, G=0, B=0)
中等边缘 (96 ~ 191)       → 绿色通道 (R=0, G=edge_pixel, B=0)
弱边缘 (< 96)             → 蓝色通道 (R=0, G=0, B=edge_pixel)
```

### 可调参数

```verilog
parameter EDGE_THRESH_HIGH = 8'd192;  // 第32行：强边缘阈值（>= 此值显示红色）
parameter EDGE_THRESH_MED  = 8'd96;   // 第33行：中等边缘阈值（>= 此值显示绿色，< 此值显示蓝色）
```

### 关键代码（第 133-139 行）

```verilog
wire strong_edge, medium_edge, weak_edge;
assign strong_edge = (edge_pixel >= EDGE_THRESH_HIGH);
assign medium_edge = (edge_pixel >= EDGE_THRESH_MED) && (edge_pixel < EDGE_THRESH_HIGH);
assign weak_edge   = (edge_pixel < EDGE_THRESH_MED);
```

### RGB 映射关系

| 条件 | R | G | B | 显示颜色 |
|------|---|---|---|----------|
| `>= 192` | edge_pixel | 0 | 0 | 红色（强边缘） |
| `96 ~ 191` | 0 | edge_pixel | 0 | 绿色（中等边缘） |
| `< 96` | 0 | 0 | edge_pixel | 蓝色（弱边缘/背景） |

### 预期现象

HDMI 显示彩色边缘图：
- **红色** = 物体轮廓主线（最强边缘）
- **绿色** = 中等强度纹理
- **蓝色** = 弱边缘或平滑区域

### 验收标准

- [ ] HDMI 显示彩色边缘图（红/绿/蓝三种颜色）
- [ ] 报告中写出 R/G/B 三通道的映射条件
- [ ] 报告中给出分级阈值

## 修改位置总结

所有扩展改动仅涉及 [hdmi_sobel_display.v](sobel_02_hdmi_sobel.srcs/sources_1/new/hdmi_sobel_display.v) 的两个位置：

| 位置 | 行号 | 内容 |
|------|------|------|
| 参数定义 | 31-33 | `DISPLAY_MODE`, `EDGE_THRESH_HIGH`, `EDGE_THRESH_MED` |
| RGB 输出映射 | 141-165 | 根据 `DISPLAY_MODE` 选择输出值 |

**以下部分完全未修改**：HDMI 时序生成（`h_cnt`/`v_cnt`）、状态机（`ST_IDLE`→`ST_RUN`→`ST_WAIT`→`ST_DISPLAY`）、`edge_mem` 读写、`image_rom`/`rgb_to_gray`/`sobel_core` 例化。
